## Keybinds

`Super+Enter` - Open Terminal

`Super+Space` - Open App Launcher (Wofi)

`Super+M` - Menu Quit, Restart, Suspend,...
 
`Super+C` - Close

`Super+F` - Fullscreen

`Super+V` - Popout

`Super+W` - Random Wallpaper
