## Keybinds

`Super+Enter` - Open Terminal

`Super+Space` - Open App Launcher

`Super+M` - Menu Quit, Restart, Suspend,...
 
`Super+Q` - Close

`Super+F` - Fullscreen

`Super+V` - Popout

`Super+W` - Random Wallpaper

`Super+Z` - Zen Browser

`Super+E` - Thunar file manager
